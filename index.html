Papyrus Patches

Papyrus patches project collect a number of fixes an improvements for Eclipse Papyrus modeling tool.

Sequence Diagrams

Currently we only provide a feature patch that fixes the sequence diagrams in Papyrus, and improving the user experience of the diagram editor.

The patch replace a substantial part of the current implementation but it is kept fully compatible with the current notation model, so the diagrams created with a non-patched version, still can be loaded without problems, always they are not broken.
